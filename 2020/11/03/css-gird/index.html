    概述

    网格布局 - Gird

    容器属性

    display: gird; 默认块级元素
    display: inline-gird; 行内元素
    - 设置网格布局后，容器子元素的float,display:inline-block;display:table-ce;;,vertical-algin & column-*等都失效
    划分行列
    display: gird;
    gird-template-columns: 100px 100px 100px; 每一列的宽度
    gird-template-rows: 100px 100px 100px; 每一行的行高
    gird-template-columns: repeat(3, 100px);
    gird-template-columns: repeat(2, 100px 20px 80px); 定义6列

    gird-template-columns: [first] 100px [line2] 200px [end];
    gird-template-rouw: [row1-start] 100px [row1-end] 100px [last-line];
    gird-template-columns: repeat(3, 20px [col-start]); == gird-template-columns: 20px [col-start] 20px [col-start] 20px [start];

    gird-template-areas

    指定的网格区域[gird-area]名称来定义网格模版，重复的网格区域名称导致内容跨越这些单元格，点号[.]代表一个空格单元
    —css
    .container{
    gird-tempalte-areas:
    “ | . | …”
    “…”;
    }
    .container{
    gird-template-columns: repeat(4, 50px);
    gird-template-rows: auto;
    gird-template-areas:
    “header header header header”
    “main mian . silder”
    “footer footer footer footer”;
    }

    auto-fill 每一行容纳auto单元格，自动填充

    display: gird;
    gird-template-columns: repeat(auto-fill ,100px);

    fr

    display: gird;
    gird-template-columns: 1fr 1fr;
    - 1fr 3fr 两列
    - 150px 1fr 2fr 三列
    - 1fr 1fr minmax(100px, 1fr) minmax()长度范围，最小值，最大值
    - 100px auto 100px

    网格间距

    行间距 row-gap: 20px
    列间距 column-gap: 20px
    gap: 20px 20px;

    gird-auto-flow: row | column; 容器自元素排列顺序
    - row dense 禁垦呢个填满，不出现空格

    单元格水平、垂直排列

    水平方向
    justify-items: start | end | center | stretch;
    垂直方向
    align-items: start | end | center | stretch;
    place-items:

    justify-self: start | end | center | stretch
    align-self: start | end | center | tretch
    place-self:

    整个容器排列

    水平排列
    justify-content: start | end | center | stretch | space-around sapce-between; sapce-evenly 单元格间距和单元格离边距间距相等
    垂直排列
    align-content: start | end | center | stretch | space-around | space-between;
