    • Bee安装遇到的坑

    beego工具Bee

    安装命令 go get github.com/beego/bee很简单，就是会失败，一顿百度搜索，各种方法都测试了一遍，无果，最后不知咋好的…仅供参考

    最后一顿瞎操作不知道咋好的，最后一步：

    export GO111MODULE=on
export GOPROXY=https://goproxy.io

    参考

    倒数第二步：配置 .bash_profile 不知道这一步有没有用

    1. 编辑 vim ~/.bash_profile
    2. i 进入编辑状态
    3. 修改文档内容
    4. 完成敲 esc退出，shift+[+]，输入[i],[:wq]保存并退出
    5. source ~/.bash_profile 使修改生效

    然后安装

    go get -u github.com/beego/bee

    输入 bee，成功打印出提示…
